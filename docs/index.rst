Welcome to URLographer's documentation!
=======================================

URLographer is a URL mapper for the Django web framework that supplements
the `standard Django URL dispatch 
<https://docs.djangoproject.com/en/dev/topics/http/urls/>`_. It was designed
to ease the maintenance burden of sites with large numbers of legacy URLs.
